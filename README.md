### Project Scope

This project focuses only on the **front-end layout and design** only for the Spotify homepage.  
Please note that the **music player functionality, audio controls, and backend features** have **not been implemented** in this version.  

The goal of this project is to improve HTML and CSS skills by recreating a real-world UI.

💬 About This Project
This is my first web project using only HTML and CSS. I started learning HTML on June 25th and began CSS on July 1st, 2025 — so this project represents my early learning journey.
I’ve tried to make the layout responsive by using rem units so that the design adjusts well across screen sizes. Each major section of the UI is structured into separate <div> blocks.
I gave extra attention to the top search bar and the sidebar to improve their responsiveness on different devices.

Since I'm still learning, the code may not be perfectly efficient and the class names aren't always meaningful — but I’m actively working on improving these aspects as I learn more.

🛠️ Tech Stack Used
HTML & CSS only.

📚 What I Learned
While building this project, I explored a lot of things like:

How HTML tags and attributes work together to structure a page
The difference between many similar CSS properties (and when to use which)
The importance of patience and precision — sometimes it took me 30 minutes just to place one element correctly because it didn’t feel right or didn’t give the vibe I wanted 😄

This project helped me understand that front-end development is not just about code — it's also about visual sense, balance, and care. I’m still learning and improving every day!
